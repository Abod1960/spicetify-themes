## More

### About

A simple groovyfied Spicetify theme.

### Information

- To apply this theme:

  ```shell
  # for dark theme
  spicetify config current_theme Onepunch color_scheme dark
  spicetify apply

  #for light theme
  spicetify config current_theme Onepunch color_scheme light
  spicetify apply

  ```
